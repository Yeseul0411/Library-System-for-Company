<div>
  <h3>계기</h3>
</div>
<div>
도서관에 대한 접근성이 떨어지는 이용자나 
<br>
바쁜 일상 속에서 도서관을 방문하기 힘든 시민들이 쉽게 접근하기 위함.
</div>
<div>
  <h3>내용</h3>
</div>
<div>
회원<br>
●도서 검색 및 도서 정보<br>
●로그인 후 대출도서, 연체이력 조회<br>
●도서 예약<br><br>
</div>
<div>
관리자 
<br>
●도서 등록, 수정, 상태 변경(대출불가)<br>
●회원 조회
</div>
</div>

<div>
  2024.03.01 ~ 2024.04.01
</div>

<div>
  3인 개발

<div>
- 메인 화면 설계 <br>
- 도서상세페이지설계(도서목록·검색 및 대출 기능)
</div>

<div>
